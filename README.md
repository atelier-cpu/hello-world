# Hello World — Python Learning Project

A collection of beginner Python scripts for learning and practicing core language concepts.

## Files

### `src/hello.py`
A minimal introductory script. Prints a greeting using an f-string and evaluates a simple arithmetic expression.

### `src/data_types.py`
Demonstrates essential Python concepts through practical examples using people and cities as data.

## Python Concepts Covered

- **Variables and f-strings** — storing values and formatting output
- **Lists** — creating lists, accessing items by index (first, last), and getting the length
- **Dictionaries** — key-value pairs, accessing values by key
- **Lists of dictionaries** — combining both structures to represent records
- **Conditionals** — `if / elif / else` branching logic
- **Loops** — iterating over a list with `for`
- **Functions** — defining reusable functions with parameters and return values
- **Standard library** — using `random` to pick a random item, and `datetime` to get today's date

## Running the Scripts

```bash
python src/hello.py
python src/data_types.py
```
